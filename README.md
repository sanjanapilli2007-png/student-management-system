# Student Management System

## Description
This is a Python-based Student Management System developed as a first-year B.Tech project. It allows users to manage student records using a simple menu-driven interface.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save student records to a file

## Technologies Used
- Python
- Git
- GitHub

## How to Run
1. Install Python.
2. Open the project folder.
3. Run:
   python main.py
